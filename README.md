# Sales-Performance-Data-Analysis(Interactive dashboard creation using MS Excel)
## Project objective
To develop an interactive Sales Performance Dashboard in Microsoft Excel that analyzes city-wise sales data, monitors target achievement, evaluates sales executive performance, and provides meaningful business insights for effective decision-making.


## Dataset used
- <a href="https://github.com/pratiklondhe2004/Sales-performance-dashboard/blob/main/sales%20performance%20%20dataset.xlsm">Dataset</a>

## Questions(KPIs)
- What is the total sales generated by each sales executive?
- Which sales executive achieved the highest sales?
- Which sales executive achieved the lowest sales?
- What percentage of the sales target has been achieved?
- How far is each sales executive from achieving their target?
- Which sales executive is the top performer?
- Which sales executive needs performance improvement?
- How does sales performance vary across different cities?
- Which city has the best sales performance?
- What are the key sales trends and performance insights?
  
- Dashboard Interaction < a href="https://github.com/pratiklondhe2004/Sales-performance-dashboard/blob/main/dashboard%20screenshot.png">View_dashboard</a>

 ## Process
- Collected sales data from the dataset.
- Cleaned and organized the data for analysis.
- Calculated sales performance KPIs.
- Created Pivot Tables to summarize the data.
- Developed Pivot Charts for visualization.
- Added Slicers for interactive city-wise filtering.
- Designed an interactive sales dashboard layout.
- Analyzed sales performance and target achievement.
- Generated business insights from the dashboard.
- Presented the results through visual reports.
